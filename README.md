# CRUD de Cursos

Esto es una api la cual esta construida con **Node.js** y **Express**, la cual permite la gestion de cursos mediante las operaciones del c.r.u.d. (Crea, Leer, Actualizar y Eliminar), 

# Istalacion
Clonar este repositorio en tu computador
Abrir la terminal estando en la carpeta del proyecto clonado
Instalar las dependencias necesarias mediante "npm install"
Iniciar el Servidor mediante "node app.js"
